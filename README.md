# Web Shop (E-Commerce Platform)

A full-stack e-commerce platform built with Node.js/Express (Backend) and Nuxt 3 (Frontend).

## Features
- Complete Digital Store Frontend
- Integrated Payment Gateways (Xendit, Tripay, Midtrans, dll)
- Redis Caching using Upstash
- Google OAuth Login integration
- Advanced Dashboard & Analytics
- WhatsApp Bot Integration via Fonnte

## Tech Stack
- **Backend:** Node.js, Express, Socket.io, Mongoose/Redis
- **Frontend:** Nuxt 3, Vue 3, Bootstrap 5

## Getting Started
Please read the documentation in the `/TUTORIALS` directory for a complete step-by-step guide on how to set up the environment and run this project locally.

1. Install backend dependencies: `yarn install`
2. Install frontend dependencies: `cd nuxt && npm install`
3. Configure `.env` files (See `TUTORIALS/01_Environment_Setup.md`)
4. Run server: `yarn dev`
5. Run frontend: `npm run dev` in the `nuxt` folder
